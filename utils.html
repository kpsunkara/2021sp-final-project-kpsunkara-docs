Email Utils¶

This module hosts helper functions to send out email notifications.

Attributes:
LOG (logging.Logger): module level variable to hold an instance of

Logger for logging

app.utils.email.send_mail(toAdd, subject, body, *attachments)¶

Send plain text email, optionally with attachments.

Note:

Delegates to send_mail_rtf

app.utils.email.send_mail_rtf(to_add, subject, plain_body, html_body, *attachments)¶

Send rich text email with embedded objects like charts

Args:

to_add (str): one or more email addresses, separated by comma

subject (str): subject of the email

plain_body (str): plain text body of the email message.

Rendered by email readers that do not support rich text format

html_body (str): rich text body of the message. May have placeholders

to embed objects like charts. eg. <img src="cid:{%(outliers.png)s}" alt="Outliers chart"/>

attachments (list[str or os.PathLike]): one of more attachments to the message.

Spark Utils¶

Module to host pyspark related utility functions.

app.utils.spark.appId()¶

Returns the applicationId of active SparkContext

app.utils.spark.init(sc)¶

Create or initialize a SparkContext

In case the pyspark module is not imported, search the well known locations using findspark and then import pyspark

Note:

Only one instance of SparkContext is ensured.

Args:

sc (SparkContext): an instance of existing SparkContext. If None, a new SparkContext will be created with default options: app.utils.SPARK_CONFIG

Returns: None

app.utils.spark.sparkContext(sc=None)¶

Create or initialize a SparkContext and return it.

Args:

sc (SparkContext): an instance of existing SparkContext

app.utils.spark.sqlContext()¶

Create or retrieve an existing SQLContext

Returns:

An instance of SQLContext created from exiting SparkContext initialized for Hive
