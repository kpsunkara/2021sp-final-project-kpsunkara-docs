Project configuration¶

Module to host functionality related to project level configuration

Attributes:
ENV_PROFILE_NAME (str): name of the environment variable that holds the

application runtime environment as defined in ENVS

ENV_LOG_LEVEL (str): name of the environment variable that holds the

logging level. This will override the default application logging level and must match the logging levels of logging

env (ENVS): holds the environment the Application was launched in

config (dict): dictionary holding the effective Application configuration

LOG (logging.Logger): module level variable to hold an instance of

Logger for logging

class app.utils.config.ENVS(value)¶

Enumeration to define application runtime environments

Note:

ENV_PROFILE_NAME environment variable specifies one of the values

classmethod parse(val)¶

Class method to parse string into an instance of ENVS

Args:

val (str): environment name to parse

Returns:

an instance of ENVS

Raises:

ValueError: if the val is not a valid environment name
