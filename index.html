Welcome to Data Science Project Template¶

This repo serves as a template to generate a project structure and boilerplate code for Python + Spark Data Science Applications, targeting Bigdata environment. In addition to generating boilerplate code, the template also includes scripts required to build, package and deploy the app via CI/CD pipeline.
